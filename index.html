Rationale

BALRAM Bot addresses critical farming challenges like pest attacks, weed growth and soil imbalance. This affordable bot autonomously monitors crops, identifies threats using AI-based image analysis, and shares details with farmers through a web app. This ensures early detection, suggests remedies, and optimizes irrigation, thereby saving time, effort, and resources, potentially leading to a 15-20% reduction in resource use.

Robotics and Automation

Autonomous navigation using ultrasonic sensors for obstacle avoidance and movement.

IoT & Communication

ESP32-CAM captures crop data and securely sends it to the Web App.

Plant Pathology and AI

Computer Vision is utilized for real-time pest and weed classification, trained on agricultural image datasets.

ü§ñ Procedure/ Operational Description

1. Navigation and Monitoring

The bot moves autonomously through crop rows using ultrasonic sensing. The ESP32-CAM captures images which are transmitted to the web application for analysis.

2. AI Model for Pest and Weed Detection

The core innovation: The Computer Vision model classifies crop health. Early detection of pests or weeds triggers alerts and provides suggested actions via the web app, reducing yield loss.
